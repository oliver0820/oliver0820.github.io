Agroconecta - Sistema de Gestión Agrícola

ASP.NET Core MVC · SQL Server · Arquitectura ONION

Agroconecta es una aplicación web empresarial desarrollada como proyecto final del ITLA, diseñada para optimizar la gestión de fincas agrícolas mediante digitalización de procesos, control de inventario, seguimiento de cultivos y análisis de productividad.

Funcionalidades Principales:
  • Gestión de Cultivos: Registro y seguimiento por ciclo
  • Control de Insumos: Inventario de semillas, fertilizantes
  • Planificación Agrícola: Calendario de siembra/cosecha
  • Dashboard Analítico: Métricas de productividad
  • Gestión de Personal: Asignación de tareas agrícolas
  • Control Financiero: Costos, ingresos, ROI
  • Responsive Design: Acceso desde cualquier dispositivo
  • Seguridad: Autenticación y roles (Admin, Agricultor, Supervisor)
